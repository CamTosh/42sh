# 42sh

![Alt Text](./.intro.gif)

42sh is the last (and biggest) project of Epitech's first year. We had to recreate a tcsh-like shell in C. We chosed to call ours kalaSH.

## Features

### Classical features

+ Command execution.
+ Echo, env, setenv, unsetenv and cd builtins.
+ Multi-pipes.
+ && and ||
